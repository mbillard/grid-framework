Prefer without a prefix? No problem:
{% highlight scss %} @include make-grid(12, ''); // .col-1-12, .col-2-12, etc. {% endhighlight %}
Want offset, push and pull? As you wish:
{% highlight scss %} @include make-grid(12, 'sm', 'offset' 'push' 'pull'); // .col-offset-sm-3-12, .col-push-sm-4-12, .col-pull-sm-5-12 // offset comes by default, use an empty string to remove it @include make-grid(12, 'sm', ''); {% endhighlight %}
.col-md-4-12.col-push-md-8-12
.col-md-8-12.col-pull-md-4-12
.col-md-5-12.col-offset-md-1-12
.col-md-5-12.col-offset-md-1-12
Prefer 1-2 over 6-12? Enable smart class names:
{% highlight scss %} $gf-smart-class-names: true; // before the import @import "grid-framework"; @include make-grid(12, 'md'); // .col-md-1-2, .col-md-1-3, .col-md-1-4, .col-md-1 {% endhighlight %}
Nostalgic about Bootstrap 2's span prefix? Ok!
{% highlight scss %} $gf-prefix: 'span'; // before the import @import "grid-framework"; @include make-grid(12, ''); // .span-1-12, .span-2-12, .span-3-12 {% endhighlight %}
